# 【これは人間専用】リリース方法

```powershell
dotnet publish .\YukaiLarkStateTransitionDiagram\YukaiLarkStateTransitionDiagram.csproj -c Release -r win-x64 --self-contained false
```

👆　出力先はだいたいここです。  
publish という名前のフォルダーが作られて、その中に実行ファイルが入ってます。  

リリースする ZIP ファイル名へのリネーム例：  

```
YukaiLarkStateTransitionDiagram-v1.0.0-win-x64.zip
```
