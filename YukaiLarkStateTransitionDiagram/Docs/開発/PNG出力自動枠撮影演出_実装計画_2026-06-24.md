# PNG出力自動枠撮影演出 実装計画 2026-06-24

## 方針

- `Ctrl + P` を押した直後に、図全体を包むPNG出力範囲枠を自動表示する。
- ユーザーは枠内ドラッグで移動、辺・角ドラッグでサイズ調整できる。
- `Enter` でPNG保存を行い、保存成功後にシャッター風フラッシュと写真風プレビューを短く表示する。
- 保存されるPNGのポラロイド風フレームは既存の出力処理を活かす。

## 実装手順

1. `Game1.BeginPngExportSelection` で空範囲ではなく初期選択枠を作る。
2. ノードと遷移の制御点から図全体のスクリーン矩形を計算する。
3. 図が空の場合はビューポート中央に既定サイズの枠を出す。
4. 撮影成功後のフラッシュ時間と写真風プレビュー時間を管理する。
5. PNG出力モード中の下部ヘルプと使い方説明書を新しい操作に合わせる。

## 実装結果

- `Ctrl + P` 直後に、図全体を包む範囲枠が表示されるようにした。
- 空の図では中央に既定サイズの範囲枠を表示するようにした。
- 既存のドラッグ移動・辺/角リサイズ・`Alt` 中の吸着なし操作はそのまま使える。
- `Enter` でPNG保存に成功したあと、白いフラッシュと写真風プレビューを短く表示するようにした。
- 下部ヘルプと使い方説明書を、自動枠を調整して撮影する流れに更新した。
