# 既存サブステート紐づけ 実装計画 2026-07-04

## 背景

現在のサブステート操作は、通常ノード選択中の `Alt + Down` で中へ入る形になっている。

- 選択中の通常ノードに `SubstateDiagramId` があれば、そのサブステートへ入る。
- 未紐づけなら、新しいダイアグラムを作って通常ノードへ紐づける。

このため、サブステートと紐づいていない通常ノードを、既に存在するサブステート用ダイアグラムへ後から紐づける導線がない。

## 方針

短期実装では、通常ノード選択中の `Ctrl + Alt + Down` で既存ダイアグラム一覧を開き、選んだダイアグラムを選択中ノードのサブステートとして紐づける。

`Alt + Down` は今まで通り「中へ入る。なければ新規作成」に残す。`Ctrl + Alt + Down` は「既存の中身を選んで紐づける」操作にする。

将来的には、インスペクターにも次の導線を足すのが自然。

- 未紐づけ通常ノード: `新規作成` / `既存から選択`
- 紐づけ済み通常ノード: `中へ入る` / `付け替え` / `解除`

## 仕様

- 操作対象は通常ノードだけ。
- 選択中の通常ノードが未紐づけのときだけ、既存サブステート一覧を開く。
- 一覧に出す候補は、現在のダイアグラム自身を除外する。
- 既に他ノードのサブステートになっているダイアグラムは、二重親を避けるため候補から除外する。
- 現在の図へ戻る循環ができる候補は除外する。
- 紐づけは Undo/Redo 対象にする。
- 紐づけ後は、選択したサブステートへ入る。
- 候補がない場合はステータスで理由を伝える。

## 実装メモ

- `Game1` にサブステート紐づけメニュー用の状態を追加する。
- `Ctrl + Alt + Down` を `Alt + Down` より先に判定する。
- メニュー中は `Up/Down`、`Enter`、`Esc`、クリックで操作する。
- 下部ショートカットヘルプには、通常ノード選択中の候補として `Ctrl+Alt+Down` を追加する。
