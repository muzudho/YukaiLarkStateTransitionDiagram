# 【これは人間専用】メモ

わたしと Codex さんや、 Copilot さんなどの AI エージェントとやりとりするとき、  
状態遷移図がないとコミュニケーションが取れないと思ったんだぜ（＾～＾）  

そこで、わたしに入力しやすいカスタマイズがたくさんできるよう、  
状態遷移図を描く専用ツールを作ってもらおうと思ったんだぜ（＾～＾）  

フレームワークとして、 MonoGame を使って作ってほしいんだぜ（＾～＾）  


超良い感じの画面だぜ（＾▽＾）！
使い方説明書をマークダウンファイルで作ってほしいぜ（＾～＾）
リポジトリーのトップにある README ファイルはディレクトリーツリー案内板みたいなリンク集にして、
説明書を Docs フォルダー下に作ってくれだぜ（＾▽＾）！


リポジトリー直下の README.md なんだが、以下のようなセクションに分けて欲しいんだぜ（＾～＾）
